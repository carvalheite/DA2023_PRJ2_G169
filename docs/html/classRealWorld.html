Solves the Traveling Salesman Problem (TSP) for the given graph starting from a specified vertex.

This function implements a heuristic approach to solve the TSP by always choosing the nearest unvisited neighbor.

Parameters
graphThe graph representing the cities and distances between them.
startThe starting vertex for the TSP tour.
Returns
A pair containing the path (vector of vertices) and the total distance of the tour.
Exceptions
invalid_argumentif the graph is not connected.

@complexity O(V^2), where V is the number of vertices in the graph.
